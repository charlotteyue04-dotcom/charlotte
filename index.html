世界观构筑

我习惯先为故事找到一条最不可替代的规则，再让叙事、角色、对白与视觉意象都沿着它缓慢生长。世界观对我而言不是背景，而是会发光、会施压、也会悄悄改写命运的存在。

💘

关系与情绪

我偏爱书写关系里的暗流与失衡，写依赖、试探、占有、牺牲与反噬如何一点点塑造人物。比起平直的情感表达，我更在意那些危险、迟疑、克制却余震很长的东西。

CASE 01 · 悬疑 · 感官推理 · 黑暗浪漫

香判

若目光会失真，便让余香代行裁决。
以连环命案为表层叙事，以气味为真正的审讯者。每一瓶遗留在案发现场的香水，都像一张迟来的判词：既指向死者的原罪，也泄露凶手未曾熄灭的执念。

世界观 / Perfume as Verdict

旧城深巷尽头有一家不见招牌的香水铺。盲眼调香师 Elara 能从一缕残香中辨认出欲望、谎言、愧疚与恨意。接连发生的命案现场，总会留下与死者罪性相契的香气：黑樱桃的诱引、紫罗兰的嫉恨、焚雪松的冷酷、晚香玉的沉溺。于是，城中渐渐浮起一种近乎宗教般的流言——有人正以香为刃，为罪行命名。

人物弧光 / 余香返潮

最初的 Elara 只是为死者解读香气，像替沉默者发声；然而越向后走，她越无法回避一个事实——那些判词并不只写给死者，也写给她自己。她从替他人的欲望命名，走到承认自己同样被恨意、亏欠与未竟之爱悄然改写。到故事深处，香气不再只是证据，而成为命运返潮时最先抵达的海浪。

角色设计

Elara
盲眼调香师 / 余香中的审讯者

失去视觉之后，她以嗅觉重新学习世界，也以嗅觉识破人心。她冷静、寡言、近乎克制，却在每一次闻见檀香与旧木时被迫触碰最柔软也最危险的记忆。

Sophia
刑侦警官 / 灰烬上的理性者

她代表秩序，也习惯倚赖秩序，但 Elara 的存在让她开始不断越过边界。她与 Elara 之间并非简单的怀疑与合作，更像两种锋利意志在同一场风暴中互相照见。

Ethan
缺席者 / 留在香气深处的旧梦

他并不总在场，却始终借檀香、雪松、盲杖的木质触感与雨夜的气息被反复唤回。他既是 Elara 最温柔的一段往昔，也是所有真相里最不肯愈合的裂口。

CASE 02 · 奇幻 · 死后世界 · 长线守望

关于彼岸那些蝴蝶的事

名字会被剥落，思念却比记忆更早归来。
将等待写成一座真正存在的边界之城。蓝月、红海、南飞的蝶群与被篡改的名字，共同构成一套温柔而残酷的运行规则。

世界观 / The Bay Between

蝴蝶湾位于天堂与地狱之间，是亡魂短暂停泊的中转地。选择留下的人会成为引路者，名字被改写，记忆被剥离，只剩一缕无法解释的执念沉在灵魂深处。夜色降临时，蓝月升起，红海退潮，成群光蝶会掠过边界，将未说出口的话带往人间。这里最残忍之处并非遗忘，而是遗忘之后，身体仍会不由自主地朝某个方向回头。

人物弧光 / 蓝月回航

这条关系真正动人的地方，在于理性已经遗忘，情感却先一步认出。V 与 E 的靠近不是想起来所以相爱，而是尚未记起，仍会再次走向彼此。这使故事拥有一种很缓、很深、近乎潮汐般的宿命感。

角色设计

V
引路者 / 已遗忘却仍在等待的人

她表面学会了在荒凉中自我消遣，像已经与蝴蝶湾和解，事实上却反复被某种无名的潮汐感拖拽。她不知道自己在等谁，但连失眠都知道答案。

E
后来者 / 仍愿逆流而来的人

即使被告知名字会被篡改，记忆会被剥夺，他仍执拗地选择进入蝴蝶湾。那份近乎天真的笃定，为整条故事线点亮了最明亮的一处微光。

J
锋利旁观者 / 另一条未尽之线

嘴硬、冷锐、比任何人都更熟悉规则，因此也更明白规则最残忍的地方。他使这座边界之城不只容纳爱情，也容纳所有未竟执念。

CASE 03 · 都市 · 深夜关系 · 情绪氛围

月栖不眠

当月光落在阳台，她才终于被夜色完整地接住。
不依赖庞大设定，而用一个凌晨三点半的阳台写出一场极短、极近、却带着命运暗示的相遇。月光、风声、烟雾、唱到一半的旧歌，让情绪本身成为叙事空间。

场景设定 / A Balcony at 3:30 AM

这里没有宏大的异世界，只有宿舍楼阳台、风声、月亮、没来得及撤回的话与一支缓慢燃尽的烟。故事把一个微小夜晚写成完整的密室，让人短暂地从现实脱身，坠入一种介于自救与自毁之间的暧昧光晕。

人物弧光 / 月光落处

故事的转折不在于两人是否相爱，而在于 Luna 第一次把最私人、最难堪、最不愿被看见的情绪说出口，而那份倾诉没有被打断，也没有被轻视。被听见的那一刻，她像在月光里短暂地重新拥有了自己。

角色设计

Luna
深夜情绪承载者 / 被月光拖住的人

长期失眠，把秘密看得比体温还重要，却会在最不合时宜的深夜向一个几乎陌生的人缓慢崩塌。她脆弱，但那份脆弱始终带着锋利的自尊。

Ethan
月夜倾听者 / 近乎幻觉的停留

CASE 04 · 科幻 · 亲密关系控制 · 反转惊悚

完美恋人协议

爱被写成程序之后，服从看起来会比拥抱更温柔。
近未来芯片社会、黑市人格重构系统、同居关系中的控制欲与反噬。故事从一场日常争执开始，最终坠入一场关于支配、伪善与身份反写的私密实验。

世界观 / Chip, Love and Control

在芯片高度普及的近未来社会，合法植入仅允许用于记忆、学习与信息处理，任何关于情绪校正、行为修复与人格重构的技术都被明令禁止。然而，黑市系统 Perfect Like 仍在阴影中流通，承诺为使用者提供更安静、更聪明、更顺从、也更符合期待的亲密关系模板。于是，爱不再只是情感，而成了一份可以悄悄升级、重写、共享权限的协议。

人物弧光 / 权限倒置

这条故事线的华彩不在技术奇观，而在权力如何悄无声息地换手。Andrew 以为自己在进行一场人格优化，实际上却只是率先为 Mia 打开了系统核心。Mia 从被安排、被修正、被命名，走到学会反向书写协议、重新定义完美的含义。到最后，被驯化的不再是她，而是那个始终把爱误写成命令的人。

角色设计

Mia
被改造对象 / 反写协议的人

她最初看上去温顺、安静、习惯道歉，像所有会在关系里被轻易忽视的那一类人。但真正危险的恰恰是她的耐心与学习能力。她并不是突然反击，而是在沉默里一点点掌握了重写系统的方式。

Andrew
使用者 / 被日常欲望拖入深渊的人

他并不夸张，也不面目狰狞，反而普通、疲惫、平庸，自然而然地把照料、体面、顺从与爱都视作应得之物。正因为他的控制欲如此现实，这条线才显得格外冷。

产品化转译
更强调可执行性：从玩法承载，到系统文本，再到活动包装，展示内容如何真正被做成可游玩的东西。
🎮 Playable & Event Design
可游玩落地：适合做悬疑递进式剧情，通过芯片升级、权限转移、日志异常等节点逐步让玩家意识到关系中的控制权已经悄悄倒置。
系统承载：可以扩展为芯片升级日志、家庭共享终端、情绪校正协议、黑市免责声明等界面文本，让赛博设定直接进入可阅读可互动的层面。
活动方向：适合制作虚假广告 PV、终端弹窗事件、日志解谜式剧情推进与家居系统异常活动，让世界观包装和剧情反转相互增强。
